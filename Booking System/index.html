My Web Development Competencies

MARK UP (HTML5)

HTML (Hyper-Text Markup Language) is used to put the structure & content of a website together (Like a skeleton of a body).
It is the fifth and current major version of the HTML standard, and subsumes XHTML.

STYLING (CSS)

We use CSS to style and personalise HTML code to give design and flair. Without CSS, HTML would be nothing more than standard Libary classes displayed on a dull chaotic page

JAVASCRIPT(HTML5)

We use Javascript to give structure and style to a webpage similar to CSS. Java gives webpages interactive elements that engage the user by converting a static page into a interactive one
